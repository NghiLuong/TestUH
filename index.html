Browsing Guidelines

     1. Please DO NOT USE A VPN. It tends to give false results, as some websites behave differently when accessed via VPN.

     2. Use Edge or Chrome. Please give enough time for page to load completely.

     3. If page is not in your language, use browser translation (right click -> 'Translate') before judging.

     4. Pages with insecure/ not private connection – Click advanced and proceed to visit the page with insecure connection and then judge.

A page is JUNK if it is one of:

    • Auto redirect - The page automatically redirects to some other URL. Pay attention to the final URL in the pop-up window. Only exception is redirect to a login page, which is not junk.

    • Error message page - Sample error phrases - 'Video removed', 'Content not found', 'Not authorized', 'No results', 'Access Denied' etc.

    • Site is Unreachable - Could not resolve IP address / DNS resolution failure / Server took too long to respond error / Connection reset error

    • Little content - Page has almost no content (text or image), or has only ads and hence making the page useless to the user.

Below are NOT JUNK:

    • Intermittent bot / captcha check page. Clear the captcha, and judge the resulting page.

    • Login page

    • Empty social media profile (No posts)

    • Product out of stock page

    • Valid software code page.
